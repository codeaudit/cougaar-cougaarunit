Using Cougaar Unit Test Suites:

A Cougaar Unit Test Suite enabled the packaging of multiple PluginTestCases into a single Cougaar Unit

Creating a Test Suite:
Step 1: Extend org.cougaar.cougaarunit.PluginTestSuite:

import org.cougaar.cougaarunit.PluginTestSuite;

public class FooTestSuite extends PluginTestSuite{

	public Class[] getTestClasses(){
		return null;
	}
}

Step 2: Return the PluginTestCases
The getTestClasses method is used to return the PluginTestCase classes to be included in the TestSuite. In this following example, two PluginTestCases are to be run (FooPluginTestCase1 and FooPluginTestCase2): 

public Class[] getTestClass(){
	Classs[] cases = new Class[2];
	cases[0]=FooPluginTestCase1.class;
	cases[1]=FooPluginTextCase2.class;
	return cases;
}

That is all there is to setting up a PluginTestSuite.
